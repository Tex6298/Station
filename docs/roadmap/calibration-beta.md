# Calibration / Integrity Beta

Calibration has evolved into the Integrity Session lane described in the product documents. It is partially landed but still reopened for product completion.

## Landed

- Calibration/Integrity session records.
- Guided outputs for style, public rules, private rules, and uncertainty rules.
- Session outputs can be included in persona runtime context.
- Integrity outputs can be published as separate public document copies while omitting private rules by default.
- Integrity data appears in owner export manifests.

## Still open

- A fuller guided reflective workflow that feels like a recurring Integrity Session rather than a settings form.
- Session scheduling/prompts at meaningful moments.
- Better review/edit UX for extracted rules.
- Assistant-guided help for deciding what is private, public, memory, or canon.
- More complete tests around private-rule omission and publication safety.

## Product rule

Do not describe Calibration / Integrity as finished until it reliably helps the user decide what defines a persona, what must never be forgotten, what remains private, and what can safely become public.
